Política de Privacidade - AssistenteMCI

Última atualização: 07 de Dezembro de 2025

AssistenteMCI respeita sua privacidade e está comprometido em proteger seus dados pessoais. Esta política descreve como coletamos, usamos e protegemos suas informações.

1. Informações Coletadas

1.1 Gravação de Áudio

O AssistenteMCI solicita permissão para acessar o microfone do seu dispositivo para:

  • Comandos de voz: Facilitar a navegação e operação do aplicativo através de comandos de voz
  • Conversão de voz em texto (Speech-to-Text): Transformar suas instruções faladas em texto para preenchimento de formulários e registros

1.2 Como Usamos o Áudio

  • O áudio capturado é processado localmente no dispositivo
  • Nenhuma gravação de áudio é armazenada permanentemente em nossos servidores
  • O áudio é usado exclusivamente para interpretar comandos de voz e convertê-los em texto
  • Não compartilhamos gravações de áudio com terceiros
  • Você pode revogar a permissão de microfone a qualquer momento nas configurações do dispositivo

1.3 Outros Dados Coletados

  • Dados de cadastro: Nome, e-mail, telefone (necessários para uso do sistema)
  • Dados de uso: Informações sobre como você utiliza o aplicativo (para melhorias)
  • Dados de dispositivo: Modelo, versão do sistema operacional (para compatibilidade)

2. Permissões Solicitadas

O aplicativo solicita as seguintes permissões:

  • RECORD_AUDIO (Microfone): Para capturar comandos de voz e realizar conversão de fala em texto
  • INTERNET: Para comunicação com os servidores da aplicação e sincronização de dados
  • CAMERA (Câmera): Para captura de fotos e documentos quando necessário
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE: Para salvar e acessar arquivos no dispositivo

3. Base Legal para Processamento (LGPD)

Processamos seus dados pessoais com base em:

  • Consentimento: Ao aceitar esta política e conceder permissões
  • Execução de contrato: Para fornecer os serviços solicitados
  • Legítimo interesse: Para melhorar nossos serviços e experiência do usuário

4. Armazenamento e Segurança de Dados

  • Seus dados são armazenados em servidores seguros com criptografia
  • Implementamos medidas técnicas e organizacionais para proteger seus dados
  • Acesso aos dados é restrito apenas a pessoal autorizado
  • Dados de autenticação são armazenados de forma criptografada no dispositivo

5. Compartilhamento de Dados

Nós NÃO vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Quando exigido por lei ou ordem judicial
  • Com provedores de serviços essenciais (hospedagem, por exemplo) sob acordos de confidencialidade
  • Com seu consentimento explícito

6. Seus Direitos (LGPD - Lei Geral de Proteção de Dados)

Você tem os seguintes direitos sobre seus dados pessoais:

  • Acesso: Solicitar cópia dos seus dados armazenados
  • Correção: Solicitar correção de dados incorretos ou desatualizados
  • Exclusão: Solicitar a exclusão dos seus dados (direito ao esquecimento)
  • Portabilidade: Solicitar transferência dos seus dados para outro serviço
  • Revogação de consentimento: Retirar seu consentimento a qualquer momento
  • Oposição: Opor-se ao processamento dos seus dados em determinadas situações

7. Retenção de Dados

  • Dados são mantidos enquanto sua conta estiver ativa
  • Após solicitação de exclusão, dados serão removidos em até 30 dias
  • Alguns dados podem ser mantidos por período maior quando exigido por lei

8. Uso por Menores de Idade

O AssistenteMCI não é destinado a menores de 18 anos. Se você tem menos de 18 anos, não deve usar este aplicativo sem supervisão de um responsável legal.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Quando houver alterações significativas, notificaremos você através do aplicativo ou por e-mail. Recomendamos que revise esta política regularmente.

10. Cookies e Tecnologias Similares

O aplicativo pode usar tecnologias de armazenamento local para melhorar a experiência do usuário, como:

  • Preferências de configuração
  • Sessões de autenticação
  • Cache de dados para funcionamento offline

Se você não estiver satisfeito com nossas práticas de privacidade, tem o direito de apresentar uma reclamação à Autoridade Nacional de Proteção de Dados (ANPD) do Brasil.
